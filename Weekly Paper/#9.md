# 🧠 딥러닝 주요 개념 정리 (Semantic Segmentation · FCN · GAN · Diffusion)

## 🇶 Semantic Segmentation이란 무엇이며, 이미지 분류(Classification)와 어떤 차이가 있나요?

**Semantic Segmentation**은 이미지의 **모든 픽셀 단위로 클래스(label)를 예측하는 작업**이다.  
즉, 이미지의 각 픽셀이 ‘무엇을 나타내는지’를 판별한다. 예를 들어 도로 장면이라면 도로, 보행자, 자동차 등의 영역을 색상으로 구분한다.

| 구분 | 이미지 분류 (Classification) | 의미 분할 (Semantic Segmentation) |
|------|------------------------------|----------------------------------|
| 예측 단위 | 전체 이미지 | 픽셀 단위 |
| 출력 | 하나의 라벨 (예: Cat) | 픽셀별 라벨 맵 (예: Cat, Dog, Background 등) |
| 활용 예시 | 얼굴 감정 분류, 품질 검사 | 자율주행, 의료 영상 분석, 위성 이미지 분석 |

➡️ **핵심 차이**는 ‘공간적 정보 보존 여부’이다.  
Classification은 전체 이미지를 하나의 벡터로 요약하지만, Segmentation은 **픽셀 위치 정보(Spatial Information)** 를 유지한다.

## 🇶 Fully Convolutional Networks(FCN)의 주요 특징과 기존 CNN 기반 분류 모델과의 차이점은 무엇인가요?

**FCN(Fully Convolutional Network)** 은 이름 그대로, **전부 합성곱(Convolution) 연산만으로 구성된 네트워크**다.  
즉, 기존 CNN의 **완전연결층(Fully Connected Layer)** 을 제거하고, 그 대신 **업샘플링(Up-sampling)** 또는 **디컨볼루션(Deconvolution)** 층을 추가한다.

| 비교 항목 | 기존 CNN (분류용) | FCN (세그멘테이션용) |
|------------|------------------|----------------------|
| 구조 | Conv → Pooling → FC Layer | Conv → Pooling → Deconv(업샘플링) |
| 출력 형태 | 클래스 확률 (1차원 벡터) | 픽셀별 클래스 확률 (2D 맵) |
| 공간정보 | 손실됨 | 보존됨 |
| 대표적 모델 | VGG, ResNet | FCN-8s, U-Net, DeepLab 등 |

➡️ 즉, FCN은 “이미지를 줄였다가 다시 키우는” 구조를 통해 **위치 정보가 있는 라벨맵을 예측**할 수 있게 한다.

## 🇶 GAN에서 생성자(Generator)와 판별자(Discriminator)의 역할은 각각 무엇인가요?

**GAN(Generative Adversarial Network)** 은 “적대적(Adversarial)” 구조의 생성 모델이다.  
두 네트워크가 서로 경쟁하며 발전하는 방식으로 학습된다.

| 구성 요소 | 역할 | 목표 |
|-------------|------|------|
| **Generator (G)** | 노이즈 벡터에서 ‘진짜 같은 가짜 이미지’를 생성 | Discriminator를 속이는 것이 목표 |
| **Discriminator (D)** | 입력된 이미지가 진짜인지 가짜인지 판별 | Generator의 이미지를 구별하는 것이 목표 |

🧩 학습 과정 요약  
1. Generator는 무작위 노이즈로부터 이미지를 만든다.  
2. Discriminator는 그 이미지가 진짜 데이터인지 가짜인지 판별한다.  
3. Generator는 Discriminator를 속이기 위해 점점 더 ‘진짜 같은’ 이미지를 생성하도록 학습된다.  

➡️ 결과적으로, 두 모델이 경쟁하면서 **점점 더 정교한 이미지 생성 능력**이 생긴다.

## 🇶 Diffusion 모델이 이미지 생성에서 어떻게 활용되며, 어떤 장점이 있나요?

**Diffusion Model**은 최근 이미지 생성 분야에서 GAN을 대체할 만큼 강력한 성능을 보이는 모델이다.  
핵심 아이디어는 “이미지를 점진적으로 노이즈로 변환했다가, 반대로 복원하는 과정에서 생성하는 것”이다.

🌀 **동작 원리**
1. **Forward Process:**  
   깨끗한 이미지를 여러 단계에 걸쳐 점점 더 노이즈로 오염시킨다.  
2. **Reverse Process:**  
   완전한 노이즈에서 시작해, 노이즈를 점차 제거(de-noising)하며 새로운 이미지를 생성한다.

🎨 **장점**
- **안정적인 학습:** GAN처럼 Generator–Discriminator 경쟁 불균형 문제가 없다.  
- **고해상도·고품질 생성:** 세밀한 디테일과 구조를 잘 복원한다.  
- **다양한 조건부 생성(Conditional Generation)** 가능 (텍스트, 마스크, 스타일 등).  
- **대표 모델:** *DDPM, Stable Diffusion, Imagen, DALL·E 3* 등

➡️ Diffusion은 “노이즈 속에서 이미지를 되살리는 과정”을 학습함으로써,  
**자연스럽고 세밀한 이미지 생성의 새로운 표준**이 되었다.

---

📘 **정리**
| 개념 | 핵심 포인트 | 대표 모델 |
|------|--------------|------------|
| Semantic Segmentation | 픽셀 단위 예측 | U-Net, DeepLab |
| FCN | 완전연결층 제거 + 업샘플링 구조 | FCN-8s |
| GAN | 생성자 vs 판별자의 경쟁 | DCGAN, StyleGAN |
| Diffusion | 노이즈 제거 과정을 학습 | DDPM, Stable Diffusion |
